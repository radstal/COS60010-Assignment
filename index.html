HTTP vs HTTPS

HTTP is short for hypertext transfer protocol which allows systems to communicate with one another. It is commonly used to send data over internet. The issue of HTTP is it is not encrypted, and it can be intercepted by third parties using Man in the middle technique to accquire sensitive information. This may be addressed by employing a secure version of HTTP known as HTTPS. HTTPs is accomplished through the use of an SSL certificate, which establishes a secure encrypted connection between the web server and the web browser.

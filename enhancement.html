Pinterest Container / Masonry Layout

1.1 Description

Masonry Pattern uses a CSS grid layout to resemble the Pinterest layout. Furthermore, the quantity of columns displayed is responsive to screen sizes of 768px and 425px by specifying column count in the @media selector. Coloration with the nth-child pseudo class allows for pseudo-random colour appearance. Lastly, opacity and sizes are changed on hover by using transform, transition, and opacity properties.

  • Grid layout
  • :nth-child
  • transform
  • transition
